# Support

Atrium is pre-1.0 software. The best support channel for now is GitHub Issues.

Use issues for reproducible bugs, documentation gaps, and scoped feature
requests. For security issues, follow SECURITY.md instead of opening a public
issue.

For help requests, include:

- the commit or release
- whether you are running chat-only or with Centaur
- host OS and deployment method
- relevant logs with secrets removed
- the command that failed
